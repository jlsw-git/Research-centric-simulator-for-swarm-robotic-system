Pyqt5 installation:

1. Get Python 3.9 https://www.python.org/downloads/release/python-390/
2. Install pip manually https://www.geeksforgeeks.org/how-to-install-pip-on-windows/
3. Install pyqt5-tools using command:

pip install pyqt5-tools

4. Install opencv, pyautogui, Pillow, matplotlib

To convert .ui to .py (make sure cmd in the correct folder):
pyuic5 -x yourform.ui -o file.py
